📱 Mobile Price Prediction AIЭтот проект использует машинное обучение (Linear Regression) для предсказания стоимости мобильных телефонов на основе их технических характеристик. Включает в себя скрипты обучения и современный веб-интерфейс.🚀 Основные возможностиML Модель: Линейная регрессия, обученная на реальных данных из Cellphone.csv.Web UI: Интерактивный интерфейс на Flask с темной темой.Визуализация: Автоматическая генерация графиков точности (Real vs Predicted).Аналитика: Расчет точности $R^2$ и средней ошибки (MAE) в реальном времени.🛠 Стек технологийЯзык: Python 3.xML Библиотеки: Pandas, Scikit-learn, Matplotlib, JoblibWeb: Flask, HTML5, CSS3, JavaScript (Fetch API)📂 Структура проектаapp.py — Главный файл: Веб-сервер и интерфейс.train.py — Скрипт для обучения и сохранения модели в model.pkl.predict.py — Консольная версия для быстрого теста предсказаний.requirements.txt — Список всех необходимых библиотек.Cellphone.csv — Датасет с характеристиками устройств.🔧 Как запуститьУстановите зависимости:Bashpip install -r requirements.txt
Обучите модель (если нужно обновить model.pkl):Bashpython train.py
Запустите веб-приложение:Bashpython app.py
После запуска откройте в браузере: http://127.0.0.1:5000📊 Параметры вводаДля предсказания используются:RAM (ГБ)Battery (мАч)Internal Memory (ГБ)CPU Cores (Количество ядер)Rear Camera (Мп)Thickness (Толщина в мм)
